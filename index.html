Jetstream

data retrieval, processing and output

Jetstream is a modular data processing framework for retrieval, processing and outputting data. It is written in Python and configured using YAML.

  • License: GPL3
  • Documentation: http://jetstream.rtfd.org.

Installation:

pip install jetstream

Jetstream consists of the following packages:

  • jetstream.core
  • jetstream.cli

The following packages are under development; they will be pulled in once they are released.

  • jetstream.http - read and write via HTTP
  • jetstream.csv - read and write CSV files
  • jetstream.sql - SQL db read and write
